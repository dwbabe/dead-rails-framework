# Dead Rails Framework

A complete, production-ready train system for Roblox games.

## Features
- Rail-constrained movement (no derailing)
- Wheel rotation synced to speed
- Fuel consumption system
- Dashboard UI (speedometer, fuel gauge, distance, time)
- Smooth acceleration/deceleration/braking
- Modular easy to expand

## Demo
[Watch demo video](https://streamable.com/drcotm)

## Usage
1. Import scripts into ServerScriptService
2. Configure constants (maxSpeed, accelerationTime, fuelDrainRate)
3. Adjust rail path detection if needed

## Code Quality
- Clean, well-commented ModuleScripts
- No magic numbers (all constants configurable)
- Server-side movement, client-side smooth UI rendering

Ready for immediate integration.
